# 🌱 NayePankh Foundation — AI Awareness & Assistant Webpage

This repository contains an AI-focused web development project created as part of the Technical Internship Selection Task at NayePankh Foundation.

## 📌 Task 1: NayePankh AI Webpage

As part of the selection task, applicants were asked to create a simple AI-related webpage for NayePankh Foundation. Additional features such as AI chat interfaces, AI-powered functionalities, API integrations, and personalized user experiences were encouraged.

🤖 For this task, I developed an AI-powered awareness website using HTML, CSS, and JavaScript. The website includes an AI Chat Assistant, Memory Dashboard, Analytics Dashboard, Impact Section, Our Works Section, and Contact Page to provide an interactive and engaging experience for users.

💚 The project combines NGO awareness with AI-inspired features, demonstrating frontend development, user interface design, JavaScript functionality, and personalized user interaction.
