# 📌 Project Overview

As part of the Technical Internship Selection Task, applicants were asked to create a simple AI-related webpage for NayePankh Foundation. Additional features such as AI chat interfaces, AI-powered functionality, and personalized user experiences were encouraged.

For this task, an AI-powered awareness website was developed to provide information about NayePankh Foundation while offering interactive features for users.
